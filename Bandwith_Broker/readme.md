# Bandwidth Broker

## Getting started

### Prerequisites

* Python 3 
* [Pipenv](https://pipenv.pypa.io/en/latest/)

### Instalation

Cloning project 
    
    $ git clone https://github.com/TheoFontana/BE_QoS/tree/main/Bandwith_Broker

Setup the environment

    $ pipenv install

    $ pipenv install flask

    $ pipenv shell

Run the application

    (BandwidthBroker) $ python app.py
